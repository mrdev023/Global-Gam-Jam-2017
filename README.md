# GGL 2017

This project was maked when The Global Game Jam 2017 in Limoges.

## How to COMPILE

```
$ mvn package assembly:single
```

## How to RUN


```
$ java -jar target/GGL2017.jar
```

If not work try

```
$ java -XstartOnFirstThread -jar target/GGL2017.jar
```
